About This Program

This program generates new anime characters using Generative Adversarial Networks, a type of generative model composed of two neural networks: a Generator, and a Discriminator.

The network was trained on 102 passes of 8000 faces of anime characters obtained from https://www.kaggle.com/splcher/animefacedataset.

One of the biggest problems that I found with the outputs is that the images are rather similar to one another. This is a problem known as mode collapse . It is an active area of research in GANs. To fix this problem, I decided to save multiple versions of the model and then pick a random version to generate a random image from. This helped increase the variation in the generated outputs.
