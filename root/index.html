Naive Bayes MBTI Text Classifier
This program attempts to find your MBTI type given some input text. It uses the Naive Bayes algorithm to classify your MBTI type. The training data was taken from https://www.kaggle.com/datasnaek/mbti-type.

This test attempts to type you using the Cognitive Functions and not by the four-letter dichotomies.

Please note that even if this test uses the cognitive functions, it is not a very accurate test, and there are more aspects to personality type than just writing style. In order to find out your type, I recommend reading about the Cognitive Functions or watching educational Psychology videos online (Cognitive Personality Theory on YouTube.com is a great resource for this: https://www.youtube.com/channel/UCffhmXCZEpPFAUO_IbghGmg).
